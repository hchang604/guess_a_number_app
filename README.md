# Guess a Number App

## This simple guesing game allows players to:
1. Select a number
2. Start a game
3. The game's "AI" will attempt tp guess the selected number
4. With each guess, the players must provide a hint (Greater or Lower)
5. Game is over when the correct number is guessed

## Tools Used

- [react-native](https://reactnative.dev/)
- [Expo](https://expo.dev/) 
- [TypeScript](https://www.typescriptlang.org/)

# Instructions

1. `yarn`
2. `yarn start`
3. `a` - to run on andoird OR `i` to run on iOS

### Notes

- Requires a connected mobile device or emulator
